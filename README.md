# Project Description
Our project is a simulation of the Dzhanibekov Effect (also known as the Tennis Racket Theorem). The Dzhanibekov Effect is a phenomenon in which an object with principal axes of different inertia rotates stably around the first and third principal axes of rotation, but not around the second. The Dzhanibekov Effect is a result of Principal Moments of Inertia as well as the Conservation of Angular Momentum and Energy. Our simulation allows the user to experiment with how the Dzhanibekov Effect affects the rotation of wingnuts of different lengths (and therefore principal moments of inertia) about its principal axes.

# Instructions
Welcome to our simulation of the Dzhanibekov effect! To run the simulation, press the Start button. We also have a button to show/hide the three principal axes about which the wingnut rotates, called Show Principal Axes. Below is a list of simulation parameters that the user can change.

- Axis of Rotation : radio buttons; you can choose between x, y, z (by default this is y)
- Cylinder 1 Length : the length of the vertical cylinder (by default 3 meters)
- Cylinder 2 Length : the length of the horizontal cylinder (by default 5 meters)
- Angular Velocity : the angular velocity at which the wingnut rotates about the axis of rotation (by default 10 radians per second)

To reset these parameters back to default values, press the Reset button.

For best effects, stop the simulation while changing parameters (preferably not during a flip) or right after resetting the simulation.

Under the user input, we have three graphs:

- Angular Velocity Components vs Time
- Euler Angles vs Time
- Angular Momentum Components vs Time
